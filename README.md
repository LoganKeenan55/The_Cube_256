The Cube is a particle simulation built using LED matrix panels running from the 256 ESP32 maker board. It is made from 5 8x8 LED panels with a gyroscope to detect the angle of the cube, allowing for particles fall in the direction of gravity. The cube has 3D printed shell that encases the 5 different panels and leaves room for both the gyroscope and maker board. The panels have compatible 3-pin JST connectors that can easily connect to the ESP32 GIO pins. The gyroscope connects to I2C from ESP32 and allow efficient communication between the two components.
